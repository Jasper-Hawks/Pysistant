# Pysistant
A CLI Python Assistant who can complete mundane tasks for you.

## Features
* [x] Calculator (Fundamentals)
* [x] Todo list (File I/O)
* [ ] Dictonary (Working with libraries)
* [ ] Stopwatch (Fundamentals)
* [ ] Timers (Fundamentals)
